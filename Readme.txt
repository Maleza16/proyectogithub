# Rival Regions Clone

Este proyecto es un intento de recrear el juego Rival Regions para navegador web.

## Objetivo

Crear una copia jugable de Rival Regions desde cero, aprendiendo sobre desarrollo web en el proceso.

## Estructura

- `server.js`: Servidor Node.js con Express.
- `public/`: Archivos públicos (frontend: HTML, CSS, JS).
- `package.json`: Configuración del proyecto y dependencias.

## ¿Cómo empezar?

1. Instala Node.js en tu computadora.
2. Ejecuta:
   ```sh
   npm install
   npm start
   ```
3. Abre tu navegador en [http://localhost:3000](http://localhost:3000)
